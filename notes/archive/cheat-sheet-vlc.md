# Cheat Sheet VLC

## rvlc Commands

`rvlc` - Command Line Interface of VLC

| command      | function                           |
| ------------ | ---------------------------------- |
| `play`       | play                               |
| `pause`      | pause                              |
| `seek 0`     | replay                             |
| `get_title`  | see title                          |
| `get_time`   | current time in seconds / progress |
| `get_length` | total length in seconds            |
