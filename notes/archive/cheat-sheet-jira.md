# Cheat Sheet Jira

## Jira Markdown Syntax

| function     | syntax    |
| ------------ | --------- |
| link         | \`\[text  |
| bold         | `*foo*`   |
| italic       | `_foo_`   |
| list         | `-`       |
| list in list | `--`      |
| number list  | `#`       |
| headers      | `h1. h2.` |
