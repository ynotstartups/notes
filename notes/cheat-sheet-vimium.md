# Cheat Sheet Vimium

## Key Mappings

| Key  | Functions                                                            |
| ---- | -------------------------------------------------------------------- |
| `gi` | focus on first text input, replace `/` in google and youtube         |
| `gu` | go to root url, i.e. from `youtube.com/watch?v=foo` to `youtube.com` |
| `H`  | go back in history                                                   |
| `L`  | go forward in history                                                |
| `J`  | go to last tab                                                       |
| `K`  | go to next tab                                                       |
