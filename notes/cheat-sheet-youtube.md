# Cheat Sheet Youtube

## Keyboard Shortcuts

| Key         |                       |
|-------------|-----------------------|
| `<`         | speed up              |
| `>`         | slow down             |
| `/`         | go to search box      |
| `,`         | next frame when pause |
| `.`         | last frame when pause |
| `Shift+n/p` | next/previous video   |
| `i`         | miniplayer            |
| `f`         | full screen           |
| `c`         | caption               |
| `space`     | play/pause            |
| `m`         | mute/unmute           |


See more in [youtube docs](https://support.google.com/youtube/answer/7631406?hl=en-GB)
