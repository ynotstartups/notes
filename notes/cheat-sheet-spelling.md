# Spelling Cheat Sheet

## `our` OR `or`

| British | US |
| ------ | ------ |
| colour   | color   |
| flavour   | flavor   |
| humour   | humor   |

## Always use ze

`se` v.s. `ze`

| British | US |
| ------ | ------ |
| apologize or apologise   | apologize   |
| organize or organise   | organize   |
